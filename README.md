# Simple Chat Application
Application is based on Symfony Framework 3. 

##Usage
For PHP built-in server, in project directory run:

    $ php bin/console server:start

### GUI
To run GUI as user go to:

    http://localhost:8000
    
To run GUI as moderator go to:    
    
    http://localhost:8000?token=MODeratorHash2016
    
To run GUI as expert go to:    
    
    http://localhost:8000?token=EXPerthash2016
